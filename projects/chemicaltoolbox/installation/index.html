Installation instruction

ChemicalToolBoX can be installed on all common unix systems. However, it is developed on Linux and I don't have access to OSX. If you want to help to enhance the documentation, please contact me.

Prerequisites:

* Python 2.6 or above
* standard C, C++ and Fortran compiler
* Autotools
* CMake
* Java Runtime Environment (JRE)
  • Debian based systems: apt-get install build-essential gfortran cmake mercurial
  • Fedora: yum install make automake gcc gcc-c++ gcc-gfortran cmake mercurial
  • OSX (MacPorts): port install gcc cmake automake mercurial

  1. Clone the latest Galaxy platform:

    hg clone https://https://bitbucket.org/galaxy/galaxy-central/

  1. Navigate to the galaxy-central folder and update it:

    cd ~/galaxy-central
hg pull
hg update

  2. Create folders for toolshed and dependencies:

    mkdir ~/shed_tools
mkdir ~/galaxy-central/tool_deps

  3. Open universe_wsgi.ini and change the dependencies directory:

    gedit ~/galaxy-central/universe_wsgi.ini

  4. Search for tool_dependency_dir = None and change it to tool_dependency_dir = ./tool_deps

  5. (Re-)Start the galaxy daemon:

    GALAXY_RUN_ALL=1 sh run.sh --stop-daemon
GALAXY_RUN_ALL=1 sh run.sh --daemon

After launching galaxy is accessible via the browser at http://localhost:8080/.

Admin Account

  1. Register a new account

  2. Promote user to admin
    • open universe_wsgi.ini
    • search admin_users = None and change it to admin_users = YOUR_EMAIL_ADDRESS
